# sabor express
### Descrição do Problema e Objetivos
O projeto **Sabor Express** tem como objetivo otimizar o processo de entrega de pedidos em um sistema de delivery, utilizando algoritmos de inteligência artificial e grafos.  
O desafio proposto é encontrar as **rotas mais eficientes** entre restaurantes e clientes, considerando distância, tempo e custo.

### Objetivos principais:
- Reduzir o tempo de entrega por meio de rotas inteligentes;  
- Explorar diferentes algoritmos de busca (BFS, DFS, A*);  
- Comparar a eficiência de cada abordagem;
